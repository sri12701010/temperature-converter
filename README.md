The Temperature Converter is a simple web application that allows users to convert temperatures between Celsius (°C), Fahrenheit (°F), and Kelvin (K). The project is built using HTML, CSS, and JavaScript, providing a user-friendly interface for quick and accurate temperature conversions.

🚀 Features
Convert Celsius to Fahrenheit and Kelvin
Convert Fahrenheit to Celsius and Kelvin
Convert Kelvin to Celsius and Fahrenheit
Instant calculation and display of results
Simple and responsive user interface
Input validation for accurate conversions
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and layout
JavaScript – Conversion logic and interactivity
